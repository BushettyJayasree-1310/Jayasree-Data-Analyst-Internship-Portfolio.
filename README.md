## 📁 Project Scope

## 🔹 Task 1: Data Cleaning (Sales Dataset)
- Performed data inspection and preprocessing
- Handled missing values using median and mode
- Removed duplicate records
- Converted date column into proper format
- Applied feature engineering (Year, Month, Day)
- Detected outliers using IQR method

📌 **Key Insight:**  
Clean and structured data is essential for accurate analysis.

---

## 🔹 Task 2: Exploratory Data Analysis
- Analyzed treatment effectiveness
- Compared response rates and survival outcomes
- Identified trends across different cancer stages

📌 **Key Insight:**  
Significant variation exists across treatments and patient conditions.

---

## 🔹 Task 3: Dashboard Development (Power BI)
- Built an interactive dashboard
- Included KPIs, filters, and visual insights
- Enabled dynamic data exploration

📌 **Key Insight:**  
Dashboards simplify complex data into actionable insights.

---

## 🔹 Task 4: Data Storytelling
- Created visual insights from analysis
- Explained trends in treatment effectiveness
- Highlighted survival patterns

📌 **Key Insight:**  
Treatment selection and early diagnosis play a critical role in survival outcomes.

---

## 📊 Key Insights from Analysis
- Treatment type significantly impacts survival rates
- Higher response rates correlate with better outcomes
- Survival decreases in advanced cancer stages
- FDA-approved treatments show more consistent performance

---

## 📈 Statistical Validation
- Applied ANOVA (Analysis of Variance)
- Hypothesis Testing:
  - H₀: No significant effect
  - H₁: Significant effect

📌 **Result:**  
P-value < 0.05 → Statistically significant differences found

---

## 💡 Recommendations
- Prioritize high-performing treatments
- Encourage early cancer detection
- Improve low-performing treatments
- Promote use of FDA-approved treatments

---

## 🎯 Conclusion
Data analysis reveals that treatment selection and early diagnosis significantly impact patient survival.  
Data-driven insights can support better healthcare decision-making.
